# Todo List

A simple and clean todo list app built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks

## Tech Stack

- HTML / CSS / JavaScript
- Alpine.js
- Git & GitHub

## Getting Started

```bash
npm install
npm run dev
```

## Development Tools

- Vite for development server
